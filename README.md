Our Little Archive 💙

A special interactive birthday project created for someone important.

This is not just a birthday website.

It is a digital archive of memories, conversations, friendship, and moments that deserve to be remembered.

---

Project Idea

This website is designed like a memory archive system.

The user goes through an emotional journey where old conversations, memories, friendship moments, and meaningful experiences are slowly recovered like hidden files.

The final destination leads to a birthday surprise.

---

Theme

- Blue and White Aesthetic
- Emotional and Cute Style
- Cinematic Story Experience
- Mobile First Design
- Friendship Based Memory Archive
